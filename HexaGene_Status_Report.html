Live API Endpoints
GET hexagene-app.onrender.com/api/health Health check — returns {"status": "ok"}
POST hexagene-app.onrender.com/api/analyze Core analysis engine — requires x-api-key header
POST hexagene-app.onrender.com/api/generate-key Generate API key — requires JWT Bearer token
GET hexagene-app.onrender.com/api/keys List all user API keys — requires JWT Bearer token
POST hexagene-app.onrender.com/auth/signup Register new user account
POST hexagene-app.onrender.com/auth/login Authenticate and receive JWT + API key

Example API Request — Client Integration
# Example: Call HexaGene analysis engine from any client curl -X POST "https://hexagene-app.onrender.com/api/analyze" \ -H "x-api-key: hx_your_key_here" \ -H "Content-Type: application/json" \ -d '{ "patient_data": { "crp": 1.6, "hba1c": 5.8, "albumin": 4.2, "egfr": 90, "rdw": 13.0, "uric_acid": 5.0 } }' # Response: { "success": true, "risk_score": 67.5, "axes": { "inflammatory": 68.0, "metabolic": 42.0, "structural": 84.0, "kinetic": 90.0, "redox": 35.0, "balance": 48.0 }, "s21_state": "moderate", "status": "Moderate" }
